Filtro Bloom

-Estructuras clases y métodos para la construcción del filtro 
almacenados en el archivo main.cpp

-Test de los métodos almacenados en el archivo main.cpp

Instrucciones de uso 
1. Compilar el archivo test.cpp con el comando g++ main.cpp -o test
2. Ejecutar el archivo test con el comando ./test
3. Se guardan los resultados en el archivo resultados.txt
